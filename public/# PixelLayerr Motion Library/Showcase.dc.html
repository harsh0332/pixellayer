PIXELLAYERR · MOTION LIBRARY · v2

Eight portable, prop-driven motion components for the existing PixelLayerr site — quiet at rest, alive on interaction. Every file is "use client", SSR-safe, self-contained, and respects reduced motion + touch. Peer deps and the Next.js mount note are commented at the top of each file.
